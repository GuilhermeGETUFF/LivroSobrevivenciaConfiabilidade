# **ORGANIZAÇÃO E CRONOGRAMA DA DISCIPLINA**

## **CRONOGRAMA**

Segue o cronograma da disciplina de Análise de Sobrevivência e Confiabilidade do primeiro semestre de 2026:

```{r, message=FALSE, warning=FALSE, results='asis',echo=FALSE}
library(dplyr)
library(kableExtra)

cronograma <- tribble(
  ~Data, ~Conteúdo, ~Cor,
  "09/03", "Introdução à disciplina de Análise de Sobrevivência e Confiabilidade", "lightblue",
  "11/03", "Mecanismos de Censura e truncamento. ", "lightblue",
  "16/03", "Funções Básicas em Análise de Sobrevivência e Confiabilidade", "lightblue",
  "18/03", "Estimação não Paramétrica (parte I)", "lightblue",
  "23/03", "Estimação não Paramétrica (parte II)", "lightblue",
  "25/03", "Laboratório: Estimação não Paramétrica", "lightblue",
  "30/03", "Comparando Curvas de Sobrevivência/Confiabilidade.", "lightblue",
  "01/04", "Laboratório: Comparando Curvas de Sobrevivência/Confiabilidade", "lightblue",
  "06/04", "Aula de Dúvidas e Exercícios", "lightblue",
  "08/04", "Prova 1: 2,5 pontos", "lightblue"
)

tabela <- cronograma %>%
  select(Data, Conteúdo)

kbl_out <- kbl(
  tabela,
  align = "l",
  col.names = c("Data", "Conteúdo"),
  escape = FALSE
) %>%
  kable_styling(
    full_width = FALSE,
    bootstrap_options = c("striped", "hover")
  ) %>%
  row_spec(0, bold = TRUE)

for (i in 1:nrow(tabela)) {
  kbl_out <- kbl_out %>%
    row_spec(i, background = cronograma$Cor[i])
}

kbl_out
```
